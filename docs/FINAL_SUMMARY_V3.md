## 💡 Quick Start

```bash
# Build
make build

# Test
make test

# CLI quick start
sdk system health
sdk agent list
sdk task create --description "Explore zone" --priority 150

# Simulation
cargo run --example simulation_demo -- --scenario multi_agent_coordination

# Performance testing
cargo run --example perf_test -- --load-test basic_api

# NLP control
echo "Create a high priority task for exploration" | sdk nlp parse
sdk nlp search "explore area" --limit 10

# Health check
curl http://localhost:8000/health

# Mobile app
cd mobile-dashboard
npm install
npm start

# Docker deployment
docker-compose up -d

# Kubernetes deployment
helm install sdk-operator ./kubernetes/operator-chart
```
